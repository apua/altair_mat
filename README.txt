The files' usage as below:

+ `README.txt` -- the README file in reST format
+ `publish.sh` -- the script to publish source code
+ `customized_osbp_recipe` -- recipes
+ `first_time_setup_recipe` -- recipes
+ `fetch_media` -- recipes
+ `test_data` -- Robot Framework test data
+ `common` -- the common source code for recipe
+ `odds_and_ends` -- collection of semi-finished programs
+ `issues.rst` -- list all ideas

For description of the Altair recipe, please see:
